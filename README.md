# Auto Counter with Speed Control (React)

This is a simple React application that allows manual and automatic counting with adjustable speed.

## Features

* Increment and decrement counter manually
* Automatic increment and decrement
* Speed control for automatic counting
* Stop functionality
* Proper interval handling using useEffect

## Tech Stack

* React
* Vite
* Tailwind CSS
## Usage

* Use buttons to increment or decrement manually
* Start automatic counting using the auto buttons
* Increase speed using the speed button
* Stop the counter anytime

## Notes

The project uses setInterval with proper cleanup to avoid multiple intervals and memory issues.
